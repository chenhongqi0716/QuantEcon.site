The Econometrics Society Python and Julia Workshop

This workshop is designed to be a quick start introduction to the Python and Julia programming languages for Economists. We will assume that participants have some experience with programming in Matlab, Stata or some other language but would like to know more about Python and Julia and how they are useful for research in Quantitative Economics. 

To register for the course please submit your email address to the web form **below**. 

**Note:** We will use your email address to contact you just prior to the conference with some setup instructions.
